# hm6

### Description:
    Resizing images in 32x32 and 64x64

### Create venv:
    make venv

### Run tests:
    make test

### Run linters:
    make lint

### Run formatters:
    make format

### Run program:
    1. make server
    2. make worker
    3. make up

### Run program with Docker:
    make up-docker
