# homework 7

### Description:
    Асинхронный чат. Реализована только одна общая комната для переписки, 
    в которой переписываются все пользователи. При подключении клиент может
    запросить последние 50 сообщений в чате. 

### Create venv:
    make venv

### Run tests:
    make test

### Run linters:
    make lint

### Run formatters:
    make format

### Run program:
    make up

### Run program in docker:
    make up-docker
